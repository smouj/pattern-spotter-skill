Habilidad Pattern Spotter para OpenClaw.

## Qué Resuelve
Identifica patrones ocultos en conjuntos de datos complejos.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque en seguridad primero
- Soporte para reversión

## Uso
/pattern-spotter

## Ejemplo
```
Analizar código para problemas de análisis.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
